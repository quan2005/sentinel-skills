# UI/Frontend Constitution Patterns

UI/前端相关宪法的条款模板和验证方式。

当宪法涉及 Web UI、移动端 UI、任何视觉界面设计时使用。

---

## 核心原则

### Anti-Slop Mandate（避免通用 AI 设计模式）

```
避免以下"AI 生成"特征：

BANNED 字体：
- Inter, Roboto, Arial, Helvetica 作为主字体（无意图）
- system-ui 无设计说明
- Comic Sans（显然）

BANNED 配色：
- 紫到蓝渐变 + 白色背景
- #007bff (Bootstrap 蓝)
- #6c757d (Bootstrap 灰)
- Tailwind 默认色板不变

BANNED 布局：
- 所有内容完美居中
- 统一的卡片网格（同尺寸、同间距）
- 完全对称
- 三列功能区块

BANNED 模式：
- Hero + 3-col features + Pricing + CTA
- 所有元素相同圆角
- 通用渐变按钮
- 库存照片 Hero
```

## 条款模板

### 设计意图条款

```
## 条款：设计意图

界面具有独特的视觉识别度，避免通用设计模板。

**验证方式**：
- [ ] 字体选择有意图说明（非默认）
- [ ] 配色方案有主次结构（60-30-10）
- [ ] 布局有不对称或张力设计
- [ ] 有一个 memorable 的视觉元素
```

### 响应式条款

```
## 条款：响应式适配

界面在所有设备尺寸上保持良好的可用性。

**验证方式**：
- [ ] 在 375px, 768px, 1024px, 1440px 宽度下检查
- [ ] 触摸目标 ≥ 44px
- [ ] 无水平滚动
- [ ] 关键内容在移动端不隐藏
```

### 可访问性条款

```
## 条款：可访问性

界面可被键盘和屏幕阅读器访问。

**验证方式**：
- [ ] 所有交互元素可键盘访问
- [ ] 语义化 HTML（nav, main, article）
- [ ] 图片有 alt 文本
- [ ] 对比度 ≥ 4.5:1（文本）
- [ ] 焦点状态可见
```

### 加载状态条款

```
## 条款：加载反馈

用户操作有即时的视觉反馈。

**验证方式**：
- [ ] 按钮点击有视觉反馈
- [ ] 数据加载有加载状态
- [ ] 错误状态有友好提示
- [ ] 空状态有引导
```

### 性能条款

```
## 条款：性能表现

界面交互流畅，无明显卡顿。

**验证方式**：
- [ ] 首次内容绘制 < 2s
- [ ] 交互响应 < 100ms
- [ ] 无布局抖动
- [ ] 图片优化（lazy/压缩）
```

### 一致性条款

```
## 条款：设计一致性

界面遵循统一的设计系统。

**验证方式**：
- [ ] 字体大小使用统一 scale
- [ ] 间距使用统一单位（8px 基础）
- [ ] 颜色使用设计令牌
- [ ] 相似组件有相同样式
```

## 风格方向条款模板

### Editorial/Magazine 风格
```
## 条款：编辑风格

界面采用杂志风格，具有戏剧性的留白和对比。

**验证方式**：
- [ ] 使用衬线标题字体
- [ ] 标题字号 ≥ 48px
- [ ] 有网格突破元素
- [ ] 黑白为主，一个强调色
```

### Brutalist 风格
```
## 条款：野兽派风格

界面采用原始、高对比的野兽派风格。

**验证方式**：
- [ ] 使用等宽字体或粗无衬线字体
- [ ] 高对比度配色
- [ ] 可见网格边框
- [ ] 无圆角或极小圆角
```

### Technical/Developer 风格
```
## 条款：技术风格

界面面向开发者，深色模式为主。

**验证方式**：
- [ ] 深色背景优先
- [ ] 代码使用等宽字体
- [ ] 霓虹色强调
- [ ] 极简装饰
```

---

## 快速检查

```
发布前验证：
[ ] 字体选择有意图
[ ] 配色有主次结构
[ ] 布局有不对称或张力
[ ] 响应式在断点下测试通过
[ ] 可键盘导航
[ ] 加载/错误/空状态都有
[ ] 与设计师的审美标准一致
```

---

## UI 条款 Red Flags

```
❌ "界面美观" — 主观，无法验证
❌ "现代化设计" — 模糊
❌ "使用 Tailwind" — 是工具不是目标

✅ "界面在 375-1440px 宽度下保持可用"
✅ "对比度 ≥ 4.5:1 符合 WCAG AA"
✅ "所有交互元素有 hover/focus 状态"
```
