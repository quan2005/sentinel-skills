# Skills

个人Claude Code技能集合。

## 项目结构

```
skills/
├── skills/          # 技能目录
│   └── skill-name/  # 具体技能
│       └── SKILL.md # 技能说明
├── README.md        # 项目说明
└── agent_skills_spec.md # 技能规范
```

## 创建技能

1. 在 `skills/` 目录下创建技能文件夹
2. 编写 `SKILL.md` 文档
3. 参考 [agent_skills_spec.md](./agent_skills_spec.md) 了解详细规范

## 技能规范

请查看 [agent_skills_spec.md](./agent_skills_spec.md) 了解：
- 技能目录结构
- 文档编写规范
- 开发指南
- 质量标准
