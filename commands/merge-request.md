
---
description: Create a clean, verified, well-documented merge request (MR) for code changes
---

Follow these exact steps to create a PR:

Sync (Rebase) → Clean (Lint + Build) → Verify (Tests coverage > 80%) → Docs (README.md & CLAUDE.md) → Diff (Self-Review) → Squash → Describe → git push -u origin my-branch -o merge_request.create

If any of these steps fail, ask the user for help.
